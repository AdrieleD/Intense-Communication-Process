Task: Realizar Teste
Nesta atividade são realizados todos os testes esperados para o módulo desenvolvido durante o ciclo vigente.
Disciplines: Teste
Purpose
Testar os módulos desenvolvidos.
Relationships
RolesPrimary Performer:
  • Testador
Additional Performers:
  • Gerente de Projeto
  • Programador Senior
Outputs
  • Casos de Teste
  • Registro de Teste
    Main Description

    Nesta atividade, o Testador deverá realizar e documentar todos os testes esperados em relação ao módulo desenvolvido naquele ciclo. Uma vez que o teste tenha resultado positivo, o Testador também deverá enviar o referido módulo aos Programadores Senior para que estes realizem a integração do módulo com o restante do sistema já desenvolvido.

    Uma vez feita esta integração, o Testador deverá realizar novos testes com todo o sistema desenvolvido até então, de forma a garantir sua funcionalidade.
