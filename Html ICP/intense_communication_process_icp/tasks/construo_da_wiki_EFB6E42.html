Task: Construção da Wiki
Construção de um meio de pesquisa próprio para o desenvolvimento do sistema.
Disciplines: Requisitos
Purpose
Centralizar todas as informações inmportantes acerca do desenvolvimento do projeto, de forma que haja uma fonte única e confiável de informação à todos os membros do time.
Relationships
RolesPrimary Performer:

  • Gerente de Projeto
InputsMandatory:
  • None
Optional:
  • Diagrama de Classes
  • Modelo de Casos de Uso
  • Software Requirement Specification (SRS)
    Outputs
    • Wiki
      Main Description

      Nesta atividade, o Farejador já deve começar o trabalho na Wiki, para que já sejam inseridos a especificação geral do sistema e os primeiros diagramas logo que estes estejam disponíveis.

      A Wiki também deve possuir uma aba responsável por informar os prazos para cada tarefa, uma vez que estes tenham sido decididos.
