Task: Desenvolvimento do Banco de Dados
Nesta atividade é feita a análise do sistema de forma a desenvolver um banco de dados da melhor forma possível.
Disciplines: Implementação
Purpose
Criar e desenvolver um Banco de Dados satisfatório ao sistema.
Relationships
RolesPrimary Performer:
  • Analista de Banco de Dados
Additional Performers:
  • Gerente de Projeto
Main Description
Nesta atividade, o Analista de Banco de Dados analisa os produtos de trabalho gerados na disciplina de Projeto e, a partir da tecnologia de Banco de Dados especificada, cria um Banco que seja satisfatório ao atender as necessidades do sistema e usuários.
