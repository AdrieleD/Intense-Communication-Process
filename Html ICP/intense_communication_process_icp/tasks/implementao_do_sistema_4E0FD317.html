Task: Implementação do Sistema
Nesta atividade é feita a implementação do sistema de acordo com as tecnologias especificadas.
Disciplines: Implementação
Purpose

Implementação dos módulos do sistema.

Relationships
RolesPrimary Performer:
  • Programador Junior
Additional Performers:
  • Gerente de Projeto
Main Description

Nesta atividade é realizada a implementação dos módulos e funcionalidades do sistema como um todo, de acordo com os prazos e qualidades esperadas. Para que isso seja possível, o time de Programadores Júnior deve ser divido em pequenas equipes que possuirão um líder. Todas as equipes deverão implementar o mesmo módulo, de forma que em cada release o módulo possua diversas possibilidades de implementação. Esta diversidade é esperada como uma forma de promover um código otimizado e/ou com idéias superiores aos demais.

Sugere-se que a liderança de cada equipe seja rotativa, de forma a dividir a responsabilidade entre todos os membros desta.
