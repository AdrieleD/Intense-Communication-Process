Task: Definição de Prazos
Nesta atividade deverão ser definidos os prazos acerca das atividades que serão realizadas ao longo do desenvolvimento do sistema.
Disciplines: Requisitos
Purpose
Garantir que o sistema esteja disponível dentro do prazo máximo estabelecido.
Relationships
RolesPrimary Performer:
  • Gerente de Projeto
Additional Performers:
Main Description

Esta atividade deve contemplar os prazos iniciais de todas as disciplinas do ICP, devendo fazer uma análise minuciosa da quantidade de tempo disponível e levando em consideração os prazos para estabelecimento e manutenção das ferramentas a serem utilizadas.

O Gerenciador de Projetos deve estar atento aos prazos durante toda esta atividade para que nada ultrapasse o tempo disponível, atrasando as demais atividades que deverão ser realizadas.
