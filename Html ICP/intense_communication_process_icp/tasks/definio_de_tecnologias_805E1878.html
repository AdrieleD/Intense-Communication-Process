Task: Definição de tecnologias
Nesta atividade são definidos os aspectos técnicos do sistema em desenvolvimento.
Disciplines: Projeto
Purpose
Escolher as melhores tecnologias e ferramentas que se adaptem ao sistema.
Relationships
RolesPrimary Performer:
  • Gerente de Projeto
Additional Performers:
  • Analista de Banco de Dados
  • Analista de Sistemas
  • Farejador
  • Programador Junior
  • Programador Senior
  • Testador
Main Description

Nesta atividade deve-se levar em consideração os propósitos e tarefas esperadas do sistema para que possa ser escolhida a melhor tecnologia para implementação do sistema.Esta é uma atividade que envolve todos os papéis do ICP, uma vez que as decisões tomadas impactam, diretamente em  todas as tarefas que deverão ser realizadas.

Vale lembrar que os esforços necessários para adequação do time de desenvolvimento deve  também ser levado em consideração durante a escolha de tais tecnologias. Vale lembrar também que o Farejador participa ao final desta atividade recolhendo as informações geradas.
