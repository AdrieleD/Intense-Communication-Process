Role: Analista de Banco de Dados
O Analista de Banco de Dados é responsável por desenvolver um modelo de Banco de Dados que seja adequado ao projeto.
Role Sets: Papéis
Relationships
Primary Performs
  • Desenvolvimento do Banco de Dados
Additionally Performs
  • Definição de tecnologias
Main Description
Este papel tem como objetivo fazer a análise do sistema do ponto de vista necessário para elaboração e construção de um  Banco de Dados que atenda a todas as necessidades do sistema de forma satisfatória.
