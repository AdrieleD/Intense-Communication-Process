Role: Gerente de Projeto
O Gerente de projeto é o principal responsável pelo desenvolvimento do projeto como um todo .
Role Sets: Papéis
Relationships
Primary Performs
  • Definição de Prazos
  • Definição de Releases
  • Definição de tecnologias
Additionally Performs
  • Construção da Wiki
  • Desenvolvimento do Banco de Dados
  • Implementação do Sistema
  • Levantamento e Análise de Requisitos
  • Realizar Teste
  • Refinamento de Diagramas
Main Description

Este papel é responsável principalmente pelas seguintes tarefas:

  • Repassar informações importantes para a criação do Wiki.

  • Instruir a equipe para gerar um resultado satisfatório ao final de cada ciclo.

  • Gerenciar o jogo de forma a ser possível a elaboração das tarefas no prazo determinado.

  • Definir prazos para a realização de cada disciplina do ICP.

Staffing
Skills

Uma pessoa que execute este papel necessita das seguintes habilidades:

  • Capacidades de liderança e de trabalhar em equipe.
  • Atuar ativamente no processo do início ao fim.
  • Ser comunicativo.
Assignment Approaches

Este papel é normalmente executado por um grupo pequeno de pessoas (Sugere-se três integrantes) que farão a divisão entre si de cada responsabilidade deste papel. É interessante que haja uma rotação de responsabilidades entre este grupo a cada release ou atividade para reduzir a monopolização do poder durante o desenvolvimento do sistema.
