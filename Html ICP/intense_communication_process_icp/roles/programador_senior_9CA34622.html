Role: Programador Senior
O programador Senior deve possuir prática em programação de forma que possa verificar e integrar as partes produzidas para o sistema.
Role Sets: Papéis
Relationships
Additionally Performs
  • Definição de tecnologias
  • Realizar Teste
Main Description

Cabe ao Programador Senior realizar a integração entre os produtos de trabalho desenvolvidas a cada ciclo pelos Programadores Junior e pelo Analista de Banco de Dados. Esta integração deve ser realizada de forma que a nova parte disponível seja íntegra e completa para futura integração com as demais partes.

Staffing
SkillsO Programador Senior deve ser uma pessoa que já é bem familiarizada com a linguagem de programação escolhida e com o tipo de banco de dados desenvolvido.
Assignment Approaches

Este papel pode ser realizado um time de programadores, mas é importante que todos sigam um padrão ao fazer a junção das partes para que o sistema não fique com uma aparência desconecta.
