    Main Description

    Este papel é responsável principalmente pelas seguintes tarefas:

    • Verificar quais testes necessitam ser executados
    • Implementar testes individuais
    • Preparar, executar e registrar os testes
    • Analisar os erros encontrados
    • Comunicar os resultados obtidos com a equipe

    Os testes devem ser realizados não só no produto final, mas também a cada nova iteração, quando uma nova funcionalidade é liberada.
