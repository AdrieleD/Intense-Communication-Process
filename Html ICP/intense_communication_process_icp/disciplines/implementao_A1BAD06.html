Discipline: Implementação
Esta disciplina consiste em implementar as tarefas que são delegadas de acordo com prioridades pré-estabelecidas.
Relationships
Tasks
  • Desenvolvimento do Banco de Dados
  • Implementação do Sistema
Main Description

Os desenvolvedores constroem o projeto de forma incremental. Cada atividade pode ser desenvolvida em grupos ou em pares, isso de acordo com cada atividade delegada.

O propósito desta disciplina é implementar as tarefas de acordo com a demanda. Tal implementação abrange tanto a implementação do sistema em si como a análise e desenvolvimento do Banco de Dados.
