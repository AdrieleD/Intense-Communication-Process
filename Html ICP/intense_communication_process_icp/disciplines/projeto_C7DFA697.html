Main Description

Toda a definição técnica como linguagens e tecnologias que serão utilizadas no sistema devem ser especificados nesta disciplina.

Uma vez que tais aspectos tenham sido bem definidos, é feito o refinamento de todos os produtos de trabalho gerados na disciplina anterior.

É também nesta disciplina que o Gerente de Projeto particionará o sistema em pequenas releases a serem implementadas na disciplina de Implementação.
