Tool Mentor: Trello
O Trello é uma ferramenta que funciona como quadro Kanban, possuindo vários quadros que partir deles podemos controlar e acompanhar o progresso das atividades de cada equipe.
Tool: Ferramentas
Main Description
O Trello pode ser utilizado como um meio de se disponibilizar e controlar as atividades a serem exercidas durante as releases. Além disso, também pode ser feito o acompanhamento do progresso de cada atividade através da criação de quadros representativos, bem como a autoria de cada release entregue.
