Practice: Reuniões Semanais
Reuniões onde se discute o andamento do projeto.
Purpose
Esta reunião tem como objetivo levantar as dúvidas e dificuldades encontradas durante o desenvolvimento da release.
Main Description

Esta reunião deve ser realizada semanalmente, preferencialmente às sextas, com participação do líder de cada grupo de cada disciplina, se esta o tiver, e com o gerenciador de projeto. É nesta reunião que será disponibilizado o conteúdo da nova release e serão esclarecidas eventuais dúvidas sobre esta. É também nesta reunião que será divulgado o resultado parcial do jogo.

Pede-se que um novo líder de grupo seja escolhido semanalmente para que possa haver uma maior divisão de liderança, bem como exerça o valor de Compromisso entre os desenvolvedores do projeto.
