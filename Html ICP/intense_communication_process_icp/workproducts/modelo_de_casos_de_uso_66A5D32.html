Artifact: Modelo de Casos de Uso
Um Modelo de Casos de Uso deve retratar os requisitos funcionais do sistema, bem como seu relacionamento com os usuários externos do sistema e outros casos de uso.
Work Product Kinds: Produtos de Trabalho
Purpose

Promover um maior entendimento do sistema a ser projetado.

Description
Main DescriptionO Modelo de Casos de Uso é composto pelo Diagrama de Casos de Uso e uma documentação que deve ser desenvolvida para cada um dos principais casos de uso. Esta documentação deve ser completa e bem definida, uma vez que será de suma importância para o entendimento.
Notation

Sugere-se a notação especificada na UML.

Tailoring
Impact of not havingO sistema pode não ser bem entendido, o que pode causar erros de interpretação que impactarão diretamente na qualidade e custo do sistema. Além disso, pode ser necessário muitas releases de correção de erros.
